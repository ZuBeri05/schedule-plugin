# schedule-plugin

基于 BLBot 课程表功能移植的云崽机器人插件，支持多平台课程表导入。

## ✨ 功能特性

- 📅 **WakeUp课程表导入** - 直接发送分享口令或使用指令导入
- ⭐ **星链课表导入** - 支持星链课表分享消息和命令导入（新增）
- � **拾光课表导入** - 支添持拾光课表导出的JSON格式（新增）
- 📋 **ICS日历文件导入** - 支持URL链接和文件上传两种方式
- 🔄 **连堂课自动合并** - 星链课表自动识别并合并连续课程
- 📝 **课程备注** - 为课程添加备注信息
- 🏃 **翘课标记** - 支持单节课翘课，自动识别当前/下一节课
- 📋 **请假标记** - 全天请假，与翘课独立区分
- 👥 **群友课表** - 查看群友实时课程状态，区分请假/翘课
- 👁️ **查看他人课表** - 通过@提及查看他人课表
- 🛡️ **管理员操作** - 管理员可替他人执行翘课/请假/导入等操作
- 🌍 **时区支持** - 支持不同时区的课表
- 📆 **灵活日期查询** - 支持中文日期、星期、标准格式等多种日期表达

## 📦 安装方法

### 1. 下载插件

### 使用Git安装（推荐）

请将 schedule-plugin 放置在 Yunzai-Bot 的 plugins 目录下，重启 Yunzai-Bot 后即可使用。

请使用 git 进行安装，以方便后续升级。在 Yunzai-Bot 根目录夹打开终端，运行下述指令之一

```
// 使用 GitHub
git clone --depth=1 https://github.com/ZuBeri05/schedule-plugin.git plugins/schedule-plugin

// 使用 Github加速代理（国内无法访问Github使用）
git clone --depth=1 https://wget.la/https://github.com/ZuBeri05/schedule-plugin.git plugins/schedule-plugin
```
### 手工下载安装（不推荐）

手工下载安装包，解压后将`schedule-plugin-master`更名为`schedule-plugin`，然后放置在Yunzai的plugins目录内

### 2. 安装依赖

在云崽根目录执行：

```bash
pnpm install -C plugins/schedule-plugin
```

## 📖 使用指南

### 导入课表

#### WakeUp课程表

**方式A：自动识别（推荐）**
1. 打开 WakeUp 课程表 APP
2. 点击右上角分享按钮 → 在线分享课表
3. 复制完整的分享口令
4. 直接发送给机器人

**方式B：指令导入**
```
#导入wakeup xxxxx
```

#### 星链课表 ⭐新增

**方式A：自动识别（推荐）**
1. 打开星链课表 APP
2. 分享课表，复制分享消息
3. 直接发送给机器人

**方式B：指令导入**
```
#导入星链 xxxxx
```

> 星链课表支持自定义时间段，并会自动合并连堂课程

#### 拾光课表 ⭐新增

1. 在拾光课表中导出课表为 JSON 格式
2. 发送指令 `#导入课表`
3. 上传导出的 JSON 文件

> 支持拾光课表的时间段配置和自定义时间

#### ICS日历文件

**方式A：URL导入**

从教务系统获取 .ics 文件下载链接，然后：
```
#导入ics https://example.com/schedule.ics
```

**方式B：文件上传** ⭐新增

1. 发送指令 `#导入课表`
2. 上传 .ics 日历文件

> 支持标准 iCalendar 格式，自动计算学期开始日期和周数

### 查询课表

| 指令 | 说明 |
|------|------|
| `#课表` | 查看今日课表 |
| `#课表 明天` | 查看指定日期课表 |
| `#课表@某人` | 查看他人课表 |
| `群友在上什么课` | 查看群友实时课表状态 |
| `群友在上什么课 明天` | 查看群友指定日期课表 |

### 翘课 / 请假

| 指令 | 说明 |
|------|------|
| `#翘课` | 翘当前课/下一节课（自动判断） |
| `#翘课 课程名/序号` | 翘指定课程 |
| `#翘课 列表` | 查看今日课程列表 |
| `#取消翘课 课程名/序号` | 取消翘课 |
| `#请假` | 全天请假 |
| `#取消请假` | 取消全天请假 |

> 翘课状态基于时间判断：上课时间才显示"翘课中"，未上课时显示"下一节"+ 已翘标记

### 管理员操作

管理员可在指令后加 `@某人` 替他人操作：

```
#翘课@某人
#请假@某人
#导入wakeup xxxxx@某人
#删除课表@某人
```

### 支持的日期格式

- **中文日期**：明天、后天、昨天、今天
- **星期**：周一、周二...周日
- **下周**：下周一、下周二...下周日
- **标准格式**：2024-03-15、2024/03/15

## 📊 数据存储

课表数据存储在 `data/schedule/` 目录下，每个用户一个 YAML 文件。

翘课数据以 `课程名@上课时间` 为唯一标识，同名不同时段的课程可独立翘课。
