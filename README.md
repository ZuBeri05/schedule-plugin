# 云崽课程表插件

基于 BLBot 课程表功能移植的云崽机器人插件，支持多平台课程表导入和精美的图片展示。

## ✨ 功能特性

- 📅 **WakeUp课程表导入** - 直接发送分享口令或使用指令导入
- 📄 **ICS日历文件导入** - 支持标准iCalendar格式
- 📝 **课程备注** - 为课程添加备注信息
- 🏃 **翘课标记** - 标记今日翘课状态
- 🌍 **时区支持** - 支持不同时区的课表
- 📆 **灵活日期查询** - 支持中文日期、星期、标准格式等多种日期表达

## 📦 安装方法

### 1. 下载插件

将 `yunzai-schedule-plugin` 文件夹放到云崽的 `plugins` 目录下

### 2. 安装依赖

在云崽根目录执行：

```bash
pnpm install axios node-ical -w
```

### 3. 重启云崽

重启机器人即可自动加载插件

## 📖 使用指南

### 导入课表

#### WakeUp课程表

**方式A：自动识别（推荐）**
1. 打开 WakeUp 课程表 APP
2. 点击右上角分享按钮 → 在线分享课表
3. 复制完整的分享口令
4. 直接发送给机器人

**方式B：指令导入**
```
#导入wakeup xxxxx
```

#### ICS日历文件

从教务系统获取 .ics 文件下载链接，然后：
```
#导入ics https://example.com/schedule.ics
```

### 查询课表

```bash
#课表              # 查看今日课表
#课表 明天         # 查看明天课表
#课表 2024-03-15   # 查看指定日期课表
群友在上什么课      # 查看群友课表状态
群友在上什么课 明天  # 查看群友明天课表
```

### 其他功能

```bash
#翘课              # 标记/取消今日翘课
#删除课表          # 删除课表数据
#课表帮助          # 查看帮助
```

### 支持的日期格式

- **中文日期**：明天、后天、昨天、今天
- **星期**：周一、周二...周日
- **下周**：下周一、下周二...下周日
- **标准格式**：2024-03-15、2024/03/15
